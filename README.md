# My Portfolio Website Three.js

Personal portfolio built with React, TanStack Router, Vite, Tailwind CSS, and React Three Fiber.

## Run Locally

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```

## Netlify Drag-and-Drop Build

```bash
bun run build:drop
```

Upload the generated `dist-drop` folder to Netlify Drop.
